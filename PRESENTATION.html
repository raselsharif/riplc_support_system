User Roles

RoleAccess
AdminFull access, user management
UserCreate/view own tickets
ITHandle IT tickets
UnderwritingApprove UW tickets
MISApprove MIS tickets

Ticket Workflow

  • User creates ticket with department
  • Direct IT goes to IT queue
  • UW/MIS requires approval
  • Approved → IT for processing
  • Rejected → Closed
  • IT resolves → Closed

Getting Started

# Backend

cd backend && npm install && npm run dev


# Frontend

cd frontend && npm install && npm run dev

Open: http://localhost:5173
